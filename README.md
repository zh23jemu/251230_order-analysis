# F布行出库效率提升分析项目

## 项目概述

本项目旨在通过数据分析和预测，为F布行提供出库效率提升的系统性解决方案。主要功能包括：

- 订单数据整合与清洗
- 季节性销售特点分析
- 客户下单规律分析
- 累托法则（80/20法则）分析
- EIQ分析
- SARIMA销售预测
- 交互式Web可视化展示

## 环境要求

- Python 3.12
- 虚拟环境（推荐）

## 安装步骤

将项目文件下载到本地目录。

## 运行程序

### 运行Web可视化应用

该脚本将执行完整的数据分析流程，包括：

- 数据整合与清洗
- 季节性销售分析
- 客户下单规律分析

## 功能模块详解

### 1. 数据整合与清洗

- 合并12个月的订单数据
- 处理缺失值和异常值
- 提取时间特征（月份、季度、周、小时等）

### 2. 季节性销售特点分析

- 按月/季度统计销售总量
- 分析销售趋势和季节性波动
- 可视化展示销售变化

### 3. 客户下单规律分析

- 统计客户订单频率
- 分析订单时间分布
- 识别订单高峰期

### 4. 累托法则（80/20法则）分析

- SKU分类：A类（20%核心SKU）、B类、C类
- 客户分类：A类（核心客户）、B类、C类
- 可视化累托曲线

### 5. EIQ分析

- 订单量(I)：每个订单的订货量
- 品项数(E)：每个订单包含的SKU数量
- 订货量(Q)：每个SKU的平均订货量

### 6. SARIMA销售预测

- 对核心SKU进行未来52周销售预测
- 基于时间序列的预测模型
- 可视化预测结果

### 7. Web可视化应用

- 使用Chart.js实现交互式图表
- 响应式设计，适配不同屏幕尺寸
- 数据展示清晰直观

## 出库效率提升建议

基于分析结果，项目提供了以下提升出库效率的建议：

1. 基于SKU分类的仓位优化
2. 基于客户分类的拣货策略
3. 基于时间分布的人力调度

## 常见问题及解决方案

### 1. 依赖安装失败

**问题**：安装依赖时出现错误。
**解决方案**：
- 确保Python版本为3.12
- 确保虚拟环境已正确激活
- 尝试更新pip：`python -m pip install --upgrade pip`
- 手动安装失败的依赖：`pip install 包名`

### 2. Web应用无法访问

**问题**：访问http://localhost:5000时无法打开页面。
**解决方案**：
- 检查Flask服务器是否正常运行
- 检查端口5000是否被占用
- 尝试更换端口：修改app.py中的`port`参数

### 3. 数据分析脚本运行错误

**问题**：运行order_analysis.py时出现错误。
**解决方案**：
- 确保原始数据文件`案例-附件1：订单数据.xlsx`存在且路径正确
- 检查数据格式是否正确
- 查看错误信息，针对性解决
